Github Note Taker

A desktop web app to search GitHub by userid and return the user's profile, repos, and database notes on the user. New notes are added and persisted in real time.

Future considerations:
  1. Suggestions automatically populates as the userid is entered
  2. Error messaging when userids are not found
  3. Add a spinner after the app is launched

  • CLIENT SERVER SERVICE
    React - HTML5 - CSS3
    Node/Express - Firebase (React Fire)
    Axios

We Go Too

We Go Too is an IOS app which provides a social layer on top of maps' functionality, allowing users to create routes, search for existing routes around the world and invite friends to join them on those routes, with a real time GPS and chat connection.

    React Native - HTML5 - CSS3 - Flexbox - Xcode - Nodemailer
    Heroku - Github
    Node/Express - Socket.io - SQL
    RESTful API - Google Maps API - Native Maps API
    Mocha/Chai - Require
